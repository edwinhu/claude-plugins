# Secondary Sources - Bluebook Rules 15-18

Complete guidance for citing books, articles, treatises, and other secondary sources per Bluebook 21st Edition.

## Books and Treatises (Rule 15)

### Basic Form

```
[Author(s)], [Title] [Pinpoint] ([Edition] [Year]).
```

**Example:**
```
Richard A. Posner, Economic Analysis of Law 45 (9th ed. 2014).
```

### Author Names (Rule 15.1)

**Single author:**
```
Cass R. Sunstein, The Cost-Benefit Revolution 15 (2018).
```

**Two authors:**
```
William N. Eskridge, Jr. & Philip P. Frickey, Cases and Materials on Legislation 200 (5th ed. 2014).
```

**Three or more authors:**
```
Erwin Chemerinsky et al., Constitutional Law: Principles and Policies 150 (6th ed. 2019).
```

**Institutional author:**
```
American Law Institute, Restatement (Third) of Torts § 3 (2010).
```

### Titles (Rule 15.3)

**Typeface:** SMALL CAPS for books in law review footnotes.

```
Richard A. Posner, ECONOMIC ANALYSIS OF LAW 45 (9th ed. 2014).
```

**Subtitles:** Include after colon if relevant.
```
Cass R. Sunstein, THE COST-BENEFIT REVOLUTION: THE STRUGGLE TO MAKE GOVERNMENT EFFICIENT 15 (2018).
```

### Editors and Translators (Rule 15.2)

**Edited volume:**
```
The Federalist No. 78, at 464 (Alexander Hamilton) (Clinton Rossiter ed., 1961).
```

**Translator:**
```
Max Weber, Economy and Society 100 (Guenther Roth & Claus Wittich eds., Ephraim Fischoff trans., 1968).
```

### Edition and Year (Rule 15.4)

**Always include edition if not first:**
```
Richard A. Posner, Economic Analysis of Law 45 (9th ed. 2014).
```

**First edition:** Omit edition number.
```
Cass R. Sunstein, The Cost-Benefit Revolution 15 (2018).
```

**Publisher:** Generally omit for widely available books.

### Supplements and Pocket Parts

```
3 Ronald D. Rotunda & John E. Nowak, Treatise on Constitutional Law § 18.7 (5th ed. 2012 & Supp. 2020).
```

### Multi-Volume Works

```
3 William Blackstone, Commentaries *125.
```

Star pages (*) indicate original pagination in facsimile editions.

## Periodical Materials (Rule 16)

### Law Review Articles

**Basic form:**
```
[Author], [Title], [Vol.] [Journal] [First Page], [Pinpoint] ([Year]).
```

**Example:**
```
Cass R. Sunstein, On the Expressive Function of Law, 144 U. Pa. L. Rev. 2021, 2030 (1996).
```

### Typeface for Articles

| Element | Law Review Format |
|---------|-------------------|
| Author name | Roman |
| Article title | Italics |
| Journal name | SMALL CAPS |

```
Cass R. Sunstein, *On the Expressive Function of Law*, 144 U. PA. L. REV. 2021, 2030 (1996).
```

### Journal Abbreviations (Table T13)

Always use Bluebook abbreviations from Table T13.

| Full Name | Abbreviation |
|-----------|--------------|
| Harvard Law Review | Harv. L. Rev. |
| Yale Law Journal | Yale L.J. |
| Stanford Law Review | Stan. L. Rev. |
| Columbia Law Review | Colum. L. Rev. |
| University of Pennsylvania Law Review | U. Pa. L. Rev. |

### Student-Written Pieces

**Notes and Comments:**
```
Jane Student, Note, The Future of Privacy, 100 Harv. L. Rev. 1000 (2020).
Jane Student, Comment, On Jurisdiction, 50 Stan. L. Rev. 500 (2015).
```

**Designation (Note, Comment, etc.) appears after author's name.**

### Symposium Articles

```
Cass R. Sunstein, On the Expressive Function of Law, 144 U. Pa. L. Rev. 2021 (1996) (symposium on law and social norms).
```

### Online-Only Journals

```
Author, Title, Vol. J. Name Art. No. or Page (Year), URL.
```

### Forthcoming Articles

```
Author, Title, Vol. J. Name (forthcoming Year) (manuscript at Page) (on file with [repository]).
```

## Newspapers and Magazines (Rule 16.6)

### Newspapers

**Basic form:**
```
[Author], [Title], [Newspaper], [Date], at [Page].
```

**Example:**
```
Adam Liptak, *Court Ruling Could Reshape Healthcare*, N.Y. Times, June 26, 2015, at A1.
```

**Typeface:** Newspaper name in *italics* (non-consecutively paginated periodical).

### Online Newspaper Articles

```
Adam Liptak, Court Ruling Could Reshape Healthcare, N.Y. Times (June 26, 2015), https://www.nytimes.com/...
```

### Magazines

**Weekly or monthly:**
```
Jeffrey Toobin, *The Obama Brief*, New Yorker, Oct. 27, 2014, at 24.
```

### Blogs and Online Sources (Rule 18.2.4)

```
[Author], [Title], [Blog Name] ([Date]), [URL].
```

**Example:**
```
Eugene Volokh, Free Speech and Criminal Solicitation, Volokh Conspiracy (Jan. 15, 2020), https://...
```

## Restatements (Rule 12.9.5)

### Basic Form

```
Restatement ([Series]) of [Subject] § [Section] ([Year]).
```

**Example:**
```
Restatement (Second) of Contracts § 90 (Am. L. Inst. 1981).
Restatement (Third) of Torts: Liability for Physical and Emotional Harm § 3 (Am. L. Inst. 2010).
```

### Comments and Illustrations

```
Restatement (Second) of Contracts § 90 cmt. a (Am. L. Inst. 1981).
Restatement (Second) of Contracts § 90 cmt. a, illus. 1 (Am. L. Inst. 1981).
```

## Model Codes and Standards

### Uniform Laws

```
U.C.C. § 2-201 (Am. L. Inst. & Unif. L. Comm'n 2017).
```

### Model Codes

```
Model Rules of Pro. Conduct r. 1.6 (Am. Bar Ass'n 2020).
Model Penal Code § 2.02 (Am. L. Inst. 1985).
```

## Dictionaries and Encyclopedias (Rule 15.8)

### Legal Dictionaries

```
Black's Law Dictionary 1000 (11th ed. 2019).
```

**Entry within dictionary:**
```
Contract, Black's Law Dictionary (11th ed. 2019).
```

### Legal Encyclopedias

```
17A Am. Jur. 2d Contracts § 100 (2020).
17A C.J.S. Contracts § 200 (2019).
```

### General Encyclopedias

```
Encyclopedia Britannica (15th ed. 2010).
```

Generally avoid citing Wikipedia in legal scholarship.

## Unpublished Sources (Rule 17)

### Working Papers

```
[Author], [Title] ([Institutional Series & Number], [Date]), [URL].
```

**Example:**
```
Lucian A. Bebchuk & Alma Cohen, Firms' Decisions Where to Incorporate (Harvard Law Sch. John M. Olin Ctr. for L., Econ. & Bus., Discussion Paper No. 351, 2002), https://...
```

### Dissertations and Theses

```
Author, Title (Ph.D. dissertation, Institution Year).
```

### Letters and Memoranda

```
Letter from [Author] to [Recipient] (Date) (on file with [repository]).
```

### Interviews

```
Interview with [Name], [Title], in [Location] (Date).
```

## Electronic Sources (Rule 18)

### General Principles

**Prefer print sources** when identical content is available.

**When to cite online sources:**
1. Source available only online
2. Online version is official
3. Online version is superior (e.g., hyperlinks to cited sources)

### URL Format

Include full URL. If URL is excessively long, use shortened version or archived version.

```
https://perma.cc/XXXX-XXXX
```

### Parenthetical for Last Visited

For websites likely to change:
```
[URL] (last visited [Date]).
```

## Short Forms for Secondary Sources (Rule 4)

### Id.

Use when citing same source as immediately preceding citation with no intervening cites:
```
1. Cass R. Sunstein, On the Expressive Function of Law, 144 U. Pa. L. Rev. 2021, 2030 (1996).
2. Id. at 2035.
```

### Supra

Use for sources previously cited in a footnote:
```
15. Sunstein, supra note 1, at 2040.
```

**Form:**
```
[Author's last name], supra note [#], at [Page].
```

### Hereinafter

Define in first citation for sources with long titles or multiple works by same author:
```
1. Cass R. Sunstein, On the Expressive Function of Law, 144 U. Pa. L. Rev. 2021 (1996) [hereinafter Sunstein, Expressive Function].
...
15. Sunstein, Expressive Function, supra note 1, at 2040.
```

## Special Source Types

### Briefs and Court Documents (Rule 10.8.3)

```
Brief for Petitioner at 15, Brown v. Board of Education, 347 U.S. 483 (1954) (No. 1).
```

### Transcripts

```
Transcript of Oral Argument at 25, Citizens United v. FEC, 558 U.S. 310 (2010) (No. 08-205).
```

### Speeches and Addresses

```
[Speaker], [Title] (Remarks at [Event], [Date]), [URL].
```

## Common Errors

### Article Citations

| Error | Correction |
|-------|------------|
| Missing journal abbreviation periods | Harv. L. Rev. (not Harv L Rev) |
| Wrong year placement | ...2021 (1996). (not ...2021, 1996.) |
| Missing pinpoint | Always include specific page |

### Book Citations

| Error | Correction |
|-------|------------|
| Publisher included | Generally omit publisher |
| First edition noted | Omit "1st ed." |
| Missing year | Year always required |

### Short Form Errors

| Error | Correction |
|-------|------------|
| Supra without note number | Sunstein, supra note 1 |
| Id. after intervening cite | Use supra form instead |
| Wrong author name | Last name only for supra |

## Verification Checklist

Before finalizing secondary source citations:

- [ ] Author name(s) correctly formatted
- [ ] Title matches source exactly
- [ ] Journal/publisher abbreviation follows Bluebook tables
- [ ] Volume and page numbers verified
- [ ] Edition and year included
- [ ] Pinpoint provided for specific propositions
- [ ] Typeface correct for source type
- [ ] Short forms reference correct footnote number
