# Statutory and Regulatory Citations - Bluebook Rules 12-14

Complete guidance for citing statutes, regulations, and legislative materials per Bluebook 21st Edition.

## Federal Statutes (Rule 12)

### Basic Form

```
[Title] [Code] § [Section] ([Year]).
```

**Example:**
```
42 U.S.C. § 1983 (2018).
```

### United States Code (U.S.C.)

**Preferred citation:** Use U.S.C. (official code) when available.

**Alternative citations (when U.S.C. unavailable):**
- U.S.C.A. (West)
- U.S.C.S. (LexisNexis)

**Year:** Cite to year of code edition, not year of enactment.

```
Correct:   42 U.S.C. § 1983 (2018).
Wrong:     42 U.S.C. § 1983 (1871).    ← enactment year
```

### Multiple Sections

**Consecutive sections:**
```
42 U.S.C. §§ 1983-1985 (2018).
```

**Non-consecutive sections:**
```
42 U.S.C. §§ 1983, 1985, 1988 (2018).
```

### Subsections

```
42 U.S.C. § 1983(a) (2018).
42 U.S.C. § 1983(a)(1) (2018).
42 U.S.C. § 1983(a)(1)(A) (2018).
```

### Supplements

When citing provisions in supplements:
```
42 U.S.C. § 1983 (2018 & Supp. II 2020).
```

### Popular Names

May include popular name before or after citation:
```
Civil Rights Act of 1964, 42 U.S.C. §§ 2000e to 2000e-17 (2018).
```

Or:
```
42 U.S.C. §§ 2000e to 2000e-17 (2018) (Civil Rights Act of 1964).
```

## Session Laws (Rule 12.4)

### When to Use

- New statutes not yet in U.S.C.
- Citing specific session law provisions
- Historical statutory research

### Public Laws

```
Pub. L. No. 104-193, § 103, 110 Stat. 2105, 2112 (1996).
```

Format:
```
Pub. L. No. [Congress]-[Law No.], § [Section], [Vol.] Stat. [First Page], [Pinpoint] ([Year]).
```

### Statutes at Large

```
Act of July 2, 1890, ch. 647, 26 Stat. 209.
```

## State Statutes (Rule 12.3)

### Basic Form

```
[State Code Abbrev.] [Topic] § [Section] ([Publisher] [Year]).
```

### Examples by Format

**Subject-matter codes (California):**
```
Cal. Penal Code § 187 (West 2014).
```

**Title/chapter codes (Texas):**
```
Tex. Bus. & Com. Code Ann. § 17.50 (West 2011).
```

**Single-volume codes (Delaware):**
```
Del. Code Ann. tit. 8, § 102 (2020).
```

### State Code Abbreviations (Table T1)

Each state has specific abbreviation requirements. Always consult Table T1.

| State | Official Code | Example |
|-------|---------------|---------|
| California | Cal. [Subject] Code | Cal. Civ. Code § 1542 |
| New York | N.Y. [Subject] Law | N.Y. C.P.L.R. § 3211 |
| Texas | Tex. [Subject] Code Ann. | Tex. Fam. Code Ann. § 6.001 |

### Publisher

Include publisher for unofficial codes. Omit for official codes if Table T1 indicates.

## Constitutional Provisions (Rule 11)

### Federal Constitution

```
U.S. Const. art. I, § 8, cl. 3.
U.S. Const. amend. XIV, § 1.
U.S. Const. pmbl.
```

**No date for currently effective provisions.**

### State Constitutions

```
Cal. Const. art. I, § 1.
N.Y. Const. art. VI, § 1.
```

### Amended or Repealed Provisions

```
U.S. Const. amend. XVIII (repealed 1933).
U.S. Const. art. I, § 2, cl. 3, amended by U.S. Const. amend. XIV, § 2.
```

## Federal Regulations (Rule 14.2)

### Code of Federal Regulations (C.F.R.)

```
[Title] C.F.R. § [Section] ([Year]).
```

**Example:**
```
17 C.F.R. § 240.10b-5 (2020).
```

### Federal Register

For rules not yet in C.F.R.:
```
[Vol.] Fed. Reg. [Page] ([Date]) (to be codified at [C.F.R. cite]).
```

**Example:**
```
85 Fed. Reg. 12345 (Mar. 2, 2020) (to be codified at 17 C.F.R. pt. 240).
```

### Proposed Rules

```
85 Fed. Reg. 12345 (proposed Mar. 2, 2020) (to be codified at 17 C.F.R. pt. 240).
```

## State Regulations (Rule 14.3)

Follow format in Table T1 for each state.

**Example (California):**
```
Cal. Code Regs. tit. 8, § 3203 (2020).
```

## Legislative Materials (Rule 13)

### Bills

**Federal:**
```
H.R. 1234, 116th Cong. § 2 (2020).
S. 567, 116th Cong. (2019).
```

**Enacted bills:** Cite as session laws (Pub. L. No.) or codified statutes.

### Resolutions

```
H.R. Res. 123, 116th Cong. (2020).
S. Res. 456, 116th Cong. (2019).
```

### Committee Reports

**House:**
```
H.R. Rep. No. 116-123, at 5 (2019).
```

**Senate:**
```
S. Rep. No. 116-45, at 10 (2020).
```

**Conference:**
```
H.R. Rep. No. 116-789 (Conf. Rep.), at 15 (2020).
```

### Hearings

```
Hearing Title: Hearing on [Subject] Before the [Committee], [Cong.] ([Year]) (statement of [Witness]).
```

**Example:**
```
Cybersecurity Threats: Hearing Before the S. Comm. on Homeland Sec., 116th Cong. 25 (2020) (statement of Christopher Krebs, Dir., Cybersecurity & Infrastructure Sec. Agency).
```

### Congressional Debates

**Congressional Record:**
```
[Vol.] Cong. Rec. [Page] ([Year]) (statement of [Member]).
```

**Example:**
```
165 Cong. Rec. S1234 (daily ed. Feb. 10, 2019) (statement of Sen. Smith).
```

## Short Forms for Statutes (Rule 12.10)

### Id.

```
1. 42 U.S.C. § 1983 (2018).
2. Id.
3. Id. § 1985.
```

### Section Symbol Alone

After full citation in same general discussion:
```
Section 1983 provides...
```

Or:
```
§ 1983 provides...
```

### Named Acts

```
1. Civil Rights Act of 1964, 42 U.S.C. §§ 2000e to 2000e-17 (2018).
...
15. Civil Rights Act § 703.
```

## Common Errors

### Section Symbol Errors

| Error | Correction |
|-------|------------|
| Section 1983 | § 1983 (in citations) |
| §1983 | § 1983 (space required) |
| Sec. 1983 | § 1983 |
| ss. 1983-1985 | §§ 1983-1985 |

### Code Errors

| Error | Correction |
|-------|------------|
| 42 USC § 1983 | 42 U.S.C. § 1983 |
| 42 U.S.C. §1983 | 42 U.S.C. § 1983 |
| 42 U.S.C., § 1983 | 42 U.S.C. § 1983 |

### Year Errors

| Error | Correction |
|-------|------------|
| 42 U.S.C. § 1983 | 42 U.S.C. § 1983 (2018) ← year required |
| 42 U.S.C. § 1983 (1871) | 42 U.S.C. § 1983 (2018) ← code year, not enactment |

## Administrative Agency Materials

### Agency Decisions

```
[Case Name], [Vol.] [Agency Reporter] [Page] ([Agency abbrev.] [Year]).
```

**Example (SEC):**
```
In re XYZ Corp., 50 S.E.C. 123 (1989).
```

### Advisory Opinions and Guidance

```
[Agency], [Title], [Citation or URL] ([Date]).
```

**Example:**
```
SEC, Staff Accounting Bulletin No. 99, 64 Fed. Reg. 45150 (Aug. 19, 1999).
```

## Treaties and International Agreements (Rule 21.4)

### Basic Form

```
[Name], [Parties], [Date], [Treaty Source].
```

**Example:**
```
North American Free Trade Agreement, U.S.-Can.-Mex., Dec. 17, 1992, 32 I.L.M. 289.
```

### U.S. Treaties

```
[Name], [Other Party], [Date], [U.S. Source], [I.L.M. or other source].
```

## Verification Checklist

Before finalizing any statutory citation:

- [ ] Correct code cited (U.S.C., not U.S.C.A. if available)
- [ ] Section symbol (§) used with space before number
- [ ] Double section symbol (§§) for multiple sections
- [ ] Year is code edition year, not enactment year
- [ ] Supplements cited if applicable
- [ ] Popular name included if helpful
- [ ] Publisher included for unofficial state codes
- [ ] Subsection parentheses formatted correctly

## Special Situations

### Scattered Sections

When codified in scattered sections:
```
Employee Retirement Income Security Act of 1974, Pub. L. No. 93-406, 88 Stat. 829 (codified as amended in scattered sections of 26 and 29 U.S.C.).
```

### Repealed Statutes

```
18 U.S.C. § 1030 (2018), repealed by [cite to repealing legislation].
```

### Amended Statutes

When citing prior version:
```
42 U.S.C. § 1983 (1982), amended by [cite].
```
